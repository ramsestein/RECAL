RECAL Synthetic — Cross-Domain Classification Report

This report evaluates two binary classifiers trained on synthetic_features.csv (source domain) and tests them on synthetic_transfer.csv (shifted target domain). The goal is to measure how well each model withstands the synthetic covariate shift.

Domain shift applied to transfer:
• 20 independent base columns received distinct monotonic/non-linear transforms (e.g. x³, √|x|, exp, tanh).
• Engineered features (F3, FA) were recomputed from the transformed base columns.
• Target was recomputed with the same polynomial, then binarised via sigmoid + threshold 0.5.

6. Interpretation

  • Source performance is excellent for both models (AUROC ≈ 0.94–0.96), confirming they learned the original polynomial structure.
  • Transfer performance reveals the impact of domain shift. A large AUROC / F1 drop indicates the model fails to generalise to the transformed covariate distributions.
  • The confusion matrices show whether the model is systematically over-predicting or under-predicting the positive class after the shift.
  • The calibration curves show whether predicted probabilities still match observed frequencies in the shifted domain.
